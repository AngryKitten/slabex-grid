Grid Builder

Prefix
This will prefix all SGrid class names to differentiate.
Minimum Row Height
This defines the smallest height in pixels a row of cells can have. If the content of a cell extends past this value the row will grow to compensate.
Max Row Span
Since row creation is dynamic, this defines the maximum number of rows you will need to have a cell span.
Number of Grid Columns
This defines the number of columns each row of the grid will have.
Gutter
This is the space between each individual cell in pixels.
