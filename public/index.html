Welcome to SGrid

SGrid was built using css-grid and flex-box and is thus as extensive or limited as current CSS standards allow.

Already know what you want? Visit the Grid Builder to customize and download the grid.

You have some configuration options at your disposal so you can customize the grid to fit your needs.

  • prefix: This will be added to the beginning of each SGrid class to differentiate.
    • For this example we are using 'slabex'
  • min-row-height: This sets the default height of a row in pixels which will grow to fit the content.
    • For this example we are using '100'
  • max-row-span: Since rows are added dynamically, this is defines the maximum rows you can span.
    • For this example we are using '5'
  • num-columns: This is how you define the number of columns your grid will be.
    • For this example we are using '9'
  • gutter: You can set the amount of space to have between rows and columns in pixels.
    • For this example we are using '5'

Usage

To define a section of your html as an SGrid, you will need to put the class <prefix>-grid onto the parent element.
All classes within SGrid, if no prefix is chosen, can be accessed by using the prefixless class name, such as 'grid' in this case.

<div class="slabex-grid">
...cell definitions
</div>

Spanning Columns

You define cells by adding the class slabex-col-<number> with <number> being number of columns you want the cell to span.

<div class="slabex-grid">
<div class="slabex-col-3">
This cell spans 3 columns.
</div>
<div class="slabex-col-2">
This cell spans 2 columns.
</div>
<div class="slabex-col-4">
This cell spans 4 columns.
</div>
</div>

Spanning Rows

In addition to spanning columns, you can also span rows by adding the class slabex-row-span-<number> with <number> being the number of rows you want the cell to span.

This cell spans 4 columns.
</div>

Aligning Cell Content

Cell content is positioned using flex-box so there are quite a few ways you can do positioning.
5 main classes are provided for your usage.

  1. slabex-content-top: Aligns content to the top.
  2. slabex-content-right: Aligns content to the right.
  3. slabex-content-bottom: Aligns content to the bottom.
  4. slabex-content-left: Aligns content to the left.
  5. slabex-content-center: Aligns content to the center of both axes
    • If used in conjunction with slabex-content-top/right/bottom/left it will automatically center on the unspecified axis.
    • You can center on a specific axis by using slabex-content-center-horizontal and slabex-content-center-vertical
By default the content will stack horizontally. If this doesn't fit your needs, you can add slabex-content-vertical to stack cell children vertically.

<div class="slabex-grid">
<div class="slabex-content-top slabex-content-right">
Aligned to the top right.
</div>
<div class="slabex-content-bottom slabex-content-center">
Aligned to the bottom center.
</div>
<div class="slabex-content-center">
Aligned to the center.
</div>
</div>

Adaptive Design

SGrid has support for adaptive design as well.
By default it uses the 5 most commonly seen breakpoints.
Each of these breakpoint names and values are completely customizable.

  • xs: 0px
  • sm: 576px
  • md: 768px
  • lg: 992px
  • xl: 1200px

These breakpoints can be used in the same way you add regular column classes.

<div class="slabex-grid">
<div class="slabex-col-4 slabex-col-sm-3"></div>
<div class="slabex-col-2 slabex-row-span-2 slabex-col-sm-3 slabex-row-sm-span-1">
Remember you can also change the number of rows to span.
</div>
<div class="slabex-col-3">
If you want an offset, you can create an empty cell.
</div>
<div class="slabex-col-4 slabex-col-sm-6 slabex-col-lg-5">
Column classes without breakpoints apply from 0px to the first defined breakpoint.
</div>

</div>

Downloads

Like what you see?
There are two versions of SGrid available, a precompiled version and a compiled version.
The precompiled version allows you to make any configurations to the grid you want but must be compiled before usage.
The compiled version can't be configured from the default but doesn't require any compilation in order to use it.
These are the default configurations:

$config: (
prefix: "slabex-",
min-row-height: 50,
max-row-span: 12,
num-columns: 12,
gutter: 5
);

If you're ready to use SGrid, here is the precompiled version in .scss format.
You can get the no-hassle, already compiled, default values version of the grid here.
And lastly, if you don't want to deal with compiling but what to costumize the grid for your needs, visit the Grid Builder!
